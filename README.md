# lima-client

A client library for the Lima Telemetry Project

## How to use

`npm i lima-client`

Example:

```js
const lima = require("lima");
// Simply use the provided method
//Insert your server address for the first parameter
lima("127.0.0.1");
//Telemetry data sent!
```
